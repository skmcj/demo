### 简介

模板以`electron-forge`与`webpack`构建

集成了

- [x] `react`
- [x] `typescript`
- [x] `scss`
- [x] 启动前`loading`动画
- [x] 静态资源配置的功能
- [x] 访问本地磁盘资源
- [x] `react-router`单页面应用，以`history`模式多窗口加载不同路由
- [x] `CSP`限制解除

打开即用

### 使用方式

克隆到本地直接使用

```bash
# 安装依赖
npm install
# 启动
npm start
# 打包
npm run make
```
